## Sobre o Projeto

O **Viajantes** é uma aplicação web moderna e responsiva desenvolvida para simular uma agência de viagens digital. O projeto foca em oferecer uma experiência de usuário fluida e visualmente atraente, permitindo que os usuários explorem destinos nacionais e internacionais, visualizem promoções e entrem em contato com a agência.

Este projeto foi desenvolvido originalmente como parte da avaliação do **Recode Pro 2022** e tem sido modernizado para incorporar as melhores práticas de desenvolvimento web e design de interface (UI/UX).

---

## Funcionalidades

-   **Home Page Imersiva:** Apresentação visual com destaques e busca rápida de passagens.
-   **Catálogo de Destinos:** Visualização de destinos nacionais e internacionais com cards interativos.
-   **Promoções Exclusivas:** Seção dedicada a ofertas com preços comparativos.
-   **Sistema de Contato:** Formulário integrado para envio de mensagens.
-   **Fluxo de Compra Simulado:** Interface de checkout para finalização de pacotes.
-   **Design Responsivo:** Layout adaptável para Desktops, Tablets e Dispositivos Móveis.
-   **Estilo Glassmorphism:** Interface moderna utilizando efeitos de transparência e desfoque.

---

## Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

-   **HTML5:** Estruturação semântica do conteúdo.
-   **CSS3:** Estilização avançada com variáveis, Grid e Flexbox.
-   **Bootstrap 5:** Framework para layout responsivo e componentes de interface.
-   **JavaScript:** Interatividade básica e manipulação do DOM.
-   **Bootstrap Icons:** Biblioteca de ícones vetoriais.

## Como Executar

Para visualizar o projeto em sua máquina local:

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/seu-usuario/agencia-de-viagens-viajantes.git
    ```
2.  **Navegue até a pasta do projeto:**
    ```bash
    cd agencia-de-viagens-viajantes
    ```
3.  **Abra o arquivo `index.html`** no seu navegador de preferência.
